### Filesystem [Org-Tree]

Includes styles for:

-	Directories (default)
-	Files
-	Mountpoints
	-	Disk
	-	`tmpfs`
	-	System (`/dev`, `/proc`, `/sys`, etc.)
-	Links
	-	Soft
	-	Hard
-	Worktree (`git`, `hg`, `bzr`, etc.)

## Requirements

The themes use the [Roboto](https://github.com/google/roboto) font.

## Installation

Install into a pre-existing `workspace/styles/` directory.
